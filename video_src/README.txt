VCC 2026 영상 교체 방법
========================

1) 이 폴더(video_src)에 원본 영상을 아래 이름으로 넣습니다.
   (확장자는 mp4 / mov / webm 등 무엇이든 됩니다. 아래 이름으로 "시작"만 하면 인식)

   hero_pc   →  히어로 PC용 (가로, 스크롤에 맞춰 재생되는 영상)
   hero_mo   →  히어로 모바일용 (세로, 스크롤에 맞춰 재생되는 영상)
   after     →  After VCC 배경 루프
   venue     →  장소 소개 배경 루프
   cta       →  참가신청(CTA) 배경 루프

   * 하나만 바꾸려면 그 파일만 교체하면 됩니다. 나머지는 그대로 둡니다.
   * 원본은 고화질 그대로 넣으세요. 해상도/용량은 도구가 알아서 줄입니다.

2) 상위 폴더의  영상교체.bat  을 더블클릭합니다.

3) 자동으로 최적화된 영상(hero_pc.mp4 등)이 리포 루트에 생성됩니다.
   결과 표에서 용량 / 키프레임 / 길이를 확인하세요.
   노란색 [!] 경고가 뜨면 안내 문구를 따르세요.

4) 라이브 사이트에 반영하려면 리포 루트에서:
      git add *.mp4
      git commit -m "perf(video): 영상 교체"
      git push neko main
   1~2분 뒤 https://neko-yg.github.io/VCC_1/ 에서 Ctrl+Shift+R 새로고침.

주의
----
- 히어로 영상 길이를 13.6초에서 크게 바꾸면, 화면 위 텍스트
  (AI 산업 / 주식투자 변화 / 마케팅 트렌드)가 뜨는 타이밍이 어긋납니다.
  그 경우 index.html 의 timings 값(7 / 9.3 / 11.6 초)을 새 길이에 맞춰 조정하세요.
  도구가 길이 변화를 감지하면 경고로 알려줍니다.
- ffmpeg가 없으면:  winget install Gyan.FFmpeg  로 설치 후 다시 실행.
